# Creature Designer Art Checklist

Use transparent PNG or SVG layers so parts can be composited in the preview.

## 1) Core Body Layers
- `body/herbivore/body_mass_1..4`
- `body/carnivore/body_mass_1..4`
- Optional color masks for biome tinting.

## 2) Leg Sets (speed vs handling)
- `legs/set_1pair`
- `legs/set_2pair`
- `legs/set_3pair`
- `legs/set_4pair`
- `legs/set_5pair`

## 3) Tail Variants
- `tail/length_1`
- `tail/length_2`
- `tail/length_3`
- `tail/length_4`

## 4) Eye Variants
- `eyes/pairs_1`
- `eyes/pairs_2`
- `eyes/pairs_3`
- `eyes/pairs_4`

## 5) Ear Variants
- `ears/size_1`
- `ears/size_2`
- `ears/size_3`

## 6) Mouth / Jaw Variants
- `mouth/power_1`
- `mouth/power_2`
- `mouth/power_3`
- `mouth/power_4`
- Carnivore teeth overlay per level: `mouth/teeth_1..4`.

## 7) Stomach / Digestion Visuals
- `stomach/complexity_1`
- `stomach/complexity_2`
- `stomach/complexity_3`
- `stomach/complexity_4`

## 8) Fur Density Overlays
- `fur/density_0`
- `fur/density_1`
- `fur/density_2`
- `fur/density_3`
- `fur/density_4`

## 9) Horn Variants
- `horns/size_0`
- `horns/size_1`
- `horns/size_2`
- `horns/size_3`

## 10) Camouflage Overlays
- `camo/pattern_0`
- `camo/pattern_1`
- `camo/pattern_2`
- `camo/pattern_3`

## 11) Smell / Scent FX
- `smell/level_0`
- `smell/level_1`
- `smell/level_2`
- `smell/level_3`

## 12) UI Status Overlays (recommended)
- `status/hungry`
- `status/low_energy`
- `status/mature`
- `status/reproducing`
- `status/hiding`
